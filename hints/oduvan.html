Como pegar índices pares de uma lista?

Você pode iterar pela lista e verificar se o índice é par usando o operador de resto (%). Por exemplo, para somar os elementos nos índices pares de uma lista: 

valores = [1, 2, 3, 4, 5]
soma = 0
for i in range(len(valores)):
    if i % 2 == 0:
        soma = soma + valores[i]
Isso permitirá que você some apenas os elementos em índices pares.

Tem um jeito mais fácil?

Você pode usar o range com um passo de 2 para iterar apenas pelos índices pares. Isso simplifica o código e evita a necessidade de verificações dentro do loop: 

valores = [1, 2, 3, 4, 5]
soma = sum(valores[i] for i in range(0, len(valores), 2))
print(soma)  # Saída: 9
Dessa forma, você soma os elementos nos índices pares de forma mais direta.
